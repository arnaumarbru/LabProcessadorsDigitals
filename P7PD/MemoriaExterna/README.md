# Reproducción des de memoria externa
## **Introducción** 
Reproducir un archivo wave o MP3 des de una tarjeta SD con la esp32.
## **Material**
- esp32-Wroom-32D
- Software anteriormente mencionado 
- Placa de conexión de audio MAX98357 I2S
- lector tarjeta SD
- altavoz

## **Setup y funcionamiento**
Al igual que en la primera práctica se prepara el entorno de trabajo con VS studio y la extensión de PlatformIO.

A continuación implementamos el código dado en el enunciado de la práctica.

```cpp
#include "Audio.h"
#include "SD.h"
#include "FS.h"

// Digital I/O used
#define SD_CS          5
#define SPI_MOSI      23
#define SPI_MISO      19
#define SPI_SCK       18
#define I2S_DOUT      25
#define I2S_BCLK      27
#define I2S_LRC       26

Audio audio;

```
Se añaden las cabeceras correspondientes que son básicamente librerias de audio y también las que se han utilizado anteriormente para el lector de la SD. 
Se definen los pines que se usarán para conectar los componentes. Los 4 primeros servirán para conectar el lector de la SD y los 4 siguientes para el componente MAX98357. Se crea también una variable de audio.

```cpp
void setup(){
    pinMode(SD_CS, OUTPUT);
    digitalWrite(SD_CS, HIGH);
    SPI.begin(SPI_SCK, SPI_MISO, SPI_MOSI);
    Serial.begin(115200);
    SD.begin(SD_CS);
    audio.setPinout(I2S_BCLK, I2S_LRC, I2S_DOUT);
    audio.setVolume(10); // 0...21
    audio.connecttoFS(SD, "audio.wav");
}

```
En el setup se inicilizan los pines de los componentes, se modifica el volumen y se escoge el audio de la SD.
